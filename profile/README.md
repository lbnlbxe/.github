# Berkeley eXtensible Environment (BXE)

Welcome to the **B**erkeley e**X**tensible **E**nvironment (BXE) GitHub organization. Here are the links and repositories related to BXE:

- [BXE Website](https://lbnlbxe.github.io): User Guide for accessing and using the BXE Cluster hosted at Lawrence Berkeley National Lab. 
- [`lbnlbxe/bxe`](https://github.com/lbnlbxe/bxe): Scripts required for installing your own BXE
- [`lbnlcomputerarch/MoSAIC-P38`](https://github.com/lbnlcomputerarch/MoSAIC-P38): The Modular System for Acceleration Integration (MoSAIC) is a multi-tiled architecture for accurate and fast exploration of message driven computation.
- [`lbnlbxe/mosaicfiresim`](https://github.com/lbnlbxe/mosaicfiresim): Simple FireSim without Chipyard wrapper for MoSAIC.

## References

- [FireSim](https://fires.im) - FPGA-accelerated Cycle-Accurate Hardware Simulation in the Cloud
- [Chipyard](https://chipyard.readthedocs.io/en/main/) - An Agile RISC-V SoC Design Framework with in-order cores, out-of-order cores, accelerators, and more
